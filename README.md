#  Dijital Abonelik Takip Sistemi (SubTracker)

Bu proje, kullanıcıların dijital platformlardaki (Netflix, Spotify vb.) aboneliklerini takip etmelerini sağlayan bir web uygulamasıdır.

##  Proje Durumu
Şu anda projenin **temel altyapısı ve REST API katmanı** kurulmuştur. 
Katmanlı mimari (Layered Architecture) prensiplerine uygun olarak geliştirilmektedir.

##  Klasör Yapısı
- **backend/**: Sunucu tarafı kodları (Node.js, Express)
- **frontend/**: Kullanıcı arayüzü kodları (HTML, CSS, JS)
- **node_modules/**: Gerekli kütüphaneler (GitHub'a dahil edilmemiştir)
